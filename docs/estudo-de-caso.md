# Estudo de Caso – Fundamentos de DevOps

## Visão Geral
Este projeto aplica conceitos iniciais de DevOps com foco em versionamento, colaboração e automação utilizando Git e GitLab.

## Conteúdos abordados
- Fundamentos de DevOps
- Git Bash
- VS Code
- GitLab
- Pipeline CI/CD

## Aprendizados
A prática mostrou como automação, padronização e integração contínua ajudam a reduzir erros manuais e acelerar entregas.
