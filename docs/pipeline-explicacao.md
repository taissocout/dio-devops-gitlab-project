# Explicação do Pipeline CI/CD

## O que é CI/CD?
CI/CD representa Integração Contínua e Entrega Contínua, automatizando etapas importantes do ciclo de desenvolvimento.

## Estrutura deste projeto
- Build: inicia o fluxo
- Test: valida a estrutura do projeto
- Deploy: simula a entrega

## Benefícios
- automação
- mais confiabilidade
- menos erro manual
- fluxo mais organizado
