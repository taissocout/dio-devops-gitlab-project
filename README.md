# Criando seu Primeiro Projeto de DevOps com GitLab

## Descrição
Projeto desenvolvido como parte do desafio da DIO com foco em fundamentos de DevOps, versionamento com Git e automação de processos com GitLab CI/CD.

## Objetivo
Demonstrar, na prática, uma estrutura inicial de projeto DevOps utilizando documentação, versionamento e pipeline básico de CI/CD.

## Tecnologias e Ferramentas
- Git
- GitLab
- Git Bash
- VS Code
- GitLab CI/CD
- Markdown
